# Меню чата

Меню чата — точка входа в плагин из конкретной переписки. Пункты плагина
видны **только в личных чатах с людьми** — в группах, каналах и у ботов их
нет.

<Screenshot
  file="chat-menu/menu-open-light.jpg"
  dark="chat-menu/menu-open-dark.jpg"
  caption="Открытое меню чата с пунктами Streaks"
/>

| Пункт | Что делает |
| --- | --- |
| **Панель управления** | Открывает [панель управления](/control-panel) чатом: часовой пояс, синхронизация, сервисные сообщения, питомец, быстрое восстановление стрика, пересчёт и удаление. |
| **Меню восстановление стрика** | Восстановление по календарю — доступно в любое время, но не более **2 раз** на чат. → [«Стрики»](/features/streaks#2-восстановление-по-календарю--до-2-раз-на-чат-без-ограничения-по-времени) |

Второй пункт виден, только пока не исчерпан лимит восстановлений по календарю
для этого чата. Практически все остальные действия — от создания питомца до
удаления стрика — теперь собраны на [панели управления](/control-panel).
