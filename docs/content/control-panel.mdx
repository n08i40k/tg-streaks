# Панель управления

Панель управления — это единая точка входа почти для всех действий плагина в
конкретном чате: раньше они были раскиданы отдельными пунктами по [меню
чата](/chat-menu), теперь собраны в одном экране. Открывается пунктом
**«Панель управления»** в меню чата.

<Screenshot
  file="control-panel/overview-light.jpg"
  dark="control-panel/overview-dark.jpg"
  caption="Экран «Панель управления» целиком"
/>

<Callout type="info">
  Экран всегда открывается, даже если в чате ещё нет стрика или серийчика —
  часть действий на нём просто будет неактивна, пока для них нет повода
  (например, «Восстановить стрик» — пока стрик жив или не был оборван).
</Callout>

## Время

### Часовой пояс

Определяет, когда для этого чата начинаются новые сутки при подсчёте стрика —
по умолчанию используется часовой пояс устройства. Пункт открывает отдельный
экран выбора: список смещений от GMT−08:00 до GMT+08:00 с шагом в 30 минут, у
каждого — ориентировочный город и время на **вашем** устройстве, соответствующее
00:00 в этом поясе.

<Screenshot
  file="control-panel/timezone-select-light.jpg"
  dark="control-panel/timezone-select-dark.jpg"
  caption="Экран выбора часового пояса для чата"
/>

Смена часового пояса требует подтверждения — плагин предупредит, что стрик (и
серийчик, если он есть) будет пересобран заново, чтобы дни считались по новой
границе суток.

## Синхронизация

### У собеседника установлен плагин

Переключатель, которым вы отмечаете, стоит ли плагин у собеседника. От него
зависит несколько вещей:

- **Создание серийчика** — если переключатель включён, [создание
  серийчика](/features/streak-pet#как-завести-серийчика) отправляет
  собеседнику приглашение; если выключен — серийчик заводится сразу, без
  приглашения.
- **Уведомления о серийчике** — категория **«Серийчик»** в [сервисных
  сообщениях](#сервисные-сообщения) доступна только когда переключатель
  включён.

### Предложить синхронизацию

Отправляет собеседнику снимок вашей базы данных по этому чату, чтобы плагины
обеих сторон пришли к одному состоянию стрика и серийчика. Синхронизация
**односторонняя**: применив снимок, собеседник перенимает **ваше** состояние —
его собственное при этом заменяется.

Кнопка активна, только когда включён переключатель [«У собеседника установлен
плагин»](#у-собеседника-установлен-плагин) — иначе применять снимок будет
некому. После подтверждения снимок уходит собеседнику отдельным [сервисным
сообщением](/features/service-messages) **«Предложение о синхронизации!»** с
кнопкой **«Применить»**. Пока он её не нажмёт, ничего не меняется; когда
нажмёт — его стрик и серийчик перестраиваются по вашему снимку, а вам приходит
уведомление, что снимок применён.

<Callout type="info">
  Чтобы синхронизировать состояние в обе стороны, встречное предложение должен
  отправить собеседник — одно предложение выравнивает только его сторону под
  вашу.
</Callout>

## Сервисные сообщения

Пункт **«Категории сервисных сообщений»** открывает экран с тремя категориями
[сервисных сообщений](/features/service-messages#сервисные-сообщения-в-чате)
для этого чата:

| Категория | О чём сообщения |
| --- | --- |
| **Жизненный цикл** | Старт, обрыв и восстановление стрика. |
| **Улучшения** | Достижение стриком нового уровня. |
| **Серийчик** | Приглашение, переименование и удаление серийчика. |

<Screenshot
  file="control-panel/service-categories-light.jpg"
  dark="control-panel/service-categories-dark.jpg"
  caption="Экран категорий сервисных сообщений"
/>

<Callout type="info">
  Сейчас вручную переключается только категория **«Улучшения»**. «Жизненный
  цикл» включена всегда, а «Серийчик» следует за переключателем [«У собеседника
  установлен плагин»](#у-собеседника-установлен-плагин) — отдельная ручная
  настройка этих двух категорий появится позже.
</Callout>

## Серийчик

| Пункт | Что делает |
| --- | --- |
| **Создать серийчика** | Заводит [серийчика](/features/streak-pet) для этого чата. Недоступно, если серийчик уже есть. |
| **Кнопка серийчика** | Показывает или скрывает [плавающую кнопку серийчика](/features/streak-pet#плавающая-кнопка-fab) **в этом чате**. |

<Callout type="info">
  Кнопка серийчика настраивается отдельно для каждого чата. Размер самой кнопки
  общий для всех чатов и меняется в [настройках плагина](/settings#размер-кнопки).
</Callout>

## Действия

| Пункт | Что делает |
| --- | --- |
| **Восстановить стрик** | Быстро восстанавливает оборвавшийся стрик — доступно [24 часа](/features/streaks#1-быстрое-восстановление--24-часа) после обрыва. Неактивно, пока стрик жив или окно уже прошло. |
| **Перейти к началу стрика** | Открывает сообщение или день, с которого начался стрик. |

Более гибкое, но ограниченное по числу использований восстановление по
календарю осталось отдельным пунктом **«Меню восстановление стрика»** в [меню
чата](/chat-menu) — см. [«Стрики»](/features/streaks#2-восстановление-по-календарю--до-2-раз-на-чат-без-ограничения-по-времени).

## Опасная зона

<Callout type="warning">
  Все действия в этом разделе необратимы, если у вас нет [бэкапа](/backups).
  При удалении стрика серийчик удаляется вместе с ним.
</Callout>

| Пункт | Что делает |
| --- | --- |
| **Пересобрать стрика и серийчика** | Пересчитывает стрик и серийчика заново по истории переписки. См. [«Пересчёт истории»](/features/rebuild). |
| **Пересобрать серийчика** | Пересчитывает только серийчика. Доступно, если он уже создан. |
| **Удалить стрик и серийчика** | Полностью удаляет стрик и серийчика в этом чате. |
| **Удалить серийчика** | Удаляет только серийчика, стрик остаётся. Доступно, если серийчик уже создан. |

Удаление серийчика (в том числе вместе со стриком) отправляет собеседнику
[сервисное сообщение](/features/service-messages) об этом, если категория
«Серийчик» у него включена.
