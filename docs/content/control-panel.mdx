# Панель управления

Панель управления — это единая точка входа почти для всех действий плагина в
конкретном чате: раньше они были раскиданы отдельными пунктами по [меню
чата](/chat-menu), теперь собраны в одном экране. Открывается пунктом
**«Панель управления»** в меню чата.

<Screenshot
  file="control-panel/overview-light.jpg"
  dark="control-panel/overview-dark.jpg"
  caption="Экран «Панель управления» целиком"
/>

<Callout type="info">
  Экран всегда открывается, даже если в чате ещё нет стрика или питомца —
  часть действий на нём просто будет неактивна, пока для них нет повода
  (например, «Возродить стрик» — пока стрик жив или не был оборван).
</Callout>

## Время

### Часовой пояс

Определяет, когда для этого чата начинаются новые сутки при подсчёте стрика —
по умолчанию используется часовой пояс устройства. Пункт открывает отдельный
экран выбора: список смещений от GMT−08:00 до GMT+08:00 с шагом в 30 минут, у
каждого — ориентировочный город и время на **вашем** устройстве, соответствующее
00:00 в этом поясе.

<Screenshot
  file="control-panel/timezone-select-light.jpg"
  dark="control-panel/timezone-select-dark.jpg"
  caption="Экран выбора часового пояса для чата"
/>

Смена часового пояса требует подтверждения — плагин предупредит, что стрик (и
питомец, если он есть) будет пересобран заново, чтобы дни считались по новой
границе суток.

## Синхронизация

### У собеседника установлен плагин

Переключатель, которым вы отмечаете, стоит ли плагин у собеседника. От него
зависит несколько вещей:

- **Создание питомца** — если переключатель включён, [создание
  стрик-пета](/features/streak-pet#как-завести-стрик-пета) отправляет
  собеседнику приглашение; если выключен — питомец заводится сразу, без
  приглашения.
- **Уведомления о питомце** — категория **«Пет»** в [сервисных
  сообщениях](#сервисные-сообщения) доступна только когда переключатель
  включён.

### Предложить синхронизацию

<Callout type="warning">
  Кнопка пока не работает — функция синхронизации снимком стрика между
  устройствами ещё в разработке.
</Callout>

## Сервисные сообщения

Пункт **«Категории сервисных сообщений»** открывает экран с тремя категориями
[сервисных сообщений](/features/service-messages#сервисные-сообщения-в-чате)
для этого чата:

| Категория | О чём сообщения |
| --- | --- |
| **Жизненный цикл** | Старт, обрыв и восстановление стрика. |
| **Улучшения** | Достижение стриком нового уровня. |
| **Пет** | Приглашение, переименование и удаление стрик-пета. |

  настройка этих двух категорий появится позже.
</Callout>

## Питомец

| Пункт | Что делает |
| --- | --- |
| **Создать пета** | Заводит [стрик-пета](/features/streak-pet) для этого чата. Недоступно, если питомец уже есть. |
| **Кнопка питомца** | Показывает или скрывает [плавающую кнопку стрик-пета](/features/streak-pet#плавающая-кнопка-fab) **в этом чате**. |

<Callout type="info">
  Кнопка питомца настраивается отдельно для каждого чата. Размер самой кнопки
  общий для всех чатов и меняется в [настройках плагина](/settings#размер-кнопки).
</Callout>

## Действия

| Пункт | Что делает |
| --- | --- |
| **Возродить стрик** | Быстро восстанавливает оборвавшийся стрик — доступно [24 часа](/features/streaks#1-быстрое-восстановление--24-часа) после обрыва. Неактивно, пока стрик жив или окно уже прошло. |
| **Перейти к началу стрика** | Открывает сообщение или день, с которого начался стрик. |

Более гибкое, но ограниченное по числу использований восстановление по
календарю осталось отдельным пунктом **«Меню восстановление стрика»** в [меню
чата](/chat-menu) — см. [«Стрики»](/features/streaks#2-восстановление-по-календарю--до-2-раз-на-чат-без-ограничения-по-времени).

## Опасная зона

<Callout type="warning">
  Все действия в этом разделе необратимы, если у вас нет [бэкапа](/backups).
  При удалении стрика питомец удаляется вместе с ним.
</Callout>

| Пункт | Что делает |
| --- | --- |
| **Пересобрать стрика и питомца** | Пересчитывает стрик и питомца заново по истории переписки. См. [«Пересчёт истории»](/features/rebuild). |
| **Пересобрать питомца** | Пересчитывает только питомца. Доступно, если он уже создан. |
| **Удалить стрик и питомца** | Полностью удаляет стрик и питомца в этом чате. |
| **Удалить питомца** | Удаляет только питомца, стрик остаётся. Доступно, если питомец уже создан. |

Удаление питомца (в том числе вместе со стриком) отправляет собеседнику
[сервисное сообщение](/features/service-messages) об этом, если категория
«Пет» у него включена.
