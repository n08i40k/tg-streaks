# Настройки

Настройки плагина живут в разделе плагинов клиента — в карточке **Streaks**.
Их немного, и почти все продублированы более удобными точками входа (меню чата,
сервисные сообщения), так что сюда обычно заходят за бэкапами и мелкой
настройкой.

## Обновления

### Проверка обновлений

Переключатель проверки новых релизов на GitHub при старте клиента. Если
включён и вышла новая версия — внизу появится уведомление со ссылкой на канал
плагина. Сам файл при этом не скачивается автоматически — подробнее:
[«Обновления плагина»](/updates#автопроверка-новой-версии).

## Серийчик

### Размер кнопки

Размер плавающей кнопки [серийчика](/features/streak-pet#плавающая-кнопка-fab)
в чате: 64 / 80 / 96 / 112 / 128 dp. По умолчанию — 80 dp.

## Стрик

### Пересчёт стриков в личках

Массовый [пересчёт](/features/rebuild#три-варианта-запуска) стриков во всех
личных чатах с пользователями (боты и группы пропускаются). Полезно, если нужно
навести порядок сразу везде, а не в одном чате.

## Бэкапы

Три действия для управления базой плагина. Подробно они, вместе с
автоматическими бэкапами и переносом на другое устройство, описаны на странице
[«Бэкапы и перенос данных»](/backups).

### Создать бэкап

Немедленно создаёт ручную резервную копию базы плагина. Удобно перед
экспериментами, переустановкой или переносом данных.

<Screenshot
  file="settings/backup-create-light.jpg"
  dark="settings/backup-create-dark.jpg"
  caption="Кнопка «Создать бэкап»"
/>

### Восстановить бэкап

Открывает список бэкапов (или выбор файла с устройства) для восстановления.
Восстановление **полностью заменяет** текущую базу выбранной копией.

<Screenshot
  file="settings/backup-restore-light.jpg"
  dark="settings/backup-restore-dark.jpg"
  caption="Кнопка «Восстановить бэкап»"
/>

### Сбросить базу плагина

Полностью удаляет все данные плагина — стрики, серийчиков, статистику. Отменить
это нельзя, если у вас нет бэкапа.

<Screenshot
  file="settings/backup-reset-light.jpg"
  dark="settings/backup-reset-dark.jpg"
  caption="Кнопка «Сбросить базу плагина»"
/>
